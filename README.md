# Smart Railway System

This is a Railway Management Web Application.

## Developed Using
- Python Flask
- SQLite
- HTML, CSS, JavaScript

## Features
- Train booking
- Admin panel
- Passenger details management
- Ticket generation

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000
